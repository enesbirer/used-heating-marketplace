## 🚀 Proje Özellikleri

* **Modern Arayüz:** Tailwind CSS kullanılarak oluşturulmuş, hızlı ve tamamen mobil uyumlu (responsive) tasarım.
* **Admin Paneli:** Ürün ekleme, silme ve güncelleme işlemlerinin yapılabildiği şık ve kullanışlı yönetim arayüzü.
* **Çoklu Resim Desteği:** Her ürün için birden fazla fotoğraf yükleme ve detay sayfasında galeri olarak görüntüleme.
* **Akıllı Bildirim Sistemi:** İşlem sonuçlarını anında bildiren modern Toast mesajları ve özel onay pencereleri.
* **Otomatik Görsel Atama:** Resim seçilmeyen ürünlere işletmeye ait profesyonel fotoğrafların rastgele atanması.
* **WhatsApp Entegrasyonu:** Müşterilerin ürün detay sayfasından doğrudan işletme sahibiyle iletişime geçebilmesi.
* **Konum Desteği:** Google Maps entegrasyonu ile dükkan konumuna kolay erişim.

---

## 🛠️ Kullanılan Teknolojiler

* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
* **Backend:** PHP (PDO ile güvenli veritabanı bağlantısı)
* **Veritabanı:** MySQL
* **İkonlar & Fontlar:** FontAwesome, Google Fonts (Plus Jakarta Sans)

---

## 📦 Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak için:

1. Bir PHP sunucusu (XAMPP, WAMP vb.) veya Python yerleşik sunucusunu kullanabilirsiniz.

2. Python ile çalıştırmak için terminale:

```bash
python -m http.server 8000
```

3. Tarayıcıdan aşağıdaki adrese giderek uygulamayı görüntüleyebilirsiniz:

```text
http://localhost:8000
```
